四边形不等式优化

dp形式与结论

（1） dp[l][r]=min(dp[l][k]+dp[k+1][r])+w(l,r)dp[l][r]=min(dp[l][k]+dp[k+1][r])+w(l,r)dp[l][r]=min(dp[l][k]+dp[k+1][r])+w(l,r) 若dp方程满足四边形不等式，设 d[l][r]d[l][r]d[l][r] 为 dp[l][r]dp[l][r]dp[l][r] 的最优决策点，则有 d[l][r−1]≤d[l][r]≤d[l+1][r]d[l][r-1] \leq d[l][r] \leq d[l+1][r]d[l][r−1]≤d[l][r]≤d[l+1][r] （2） dp[i][j]=min(dp[i−1][k])+w(k,j)dp[i][j]=min(dp[i-1][k])+w(k,j)dp[i][j]=min(dp[i−1][k])+w(k,j) 若dp方程满足四边形不等式，设 d[i][j]d[i][j]d[i][j] 为 dp[i][j]dp[i][j]dp[i][j] 的最优决策点，则有 d[i−1][j]≤d[i][j]≤d[i][j+1]d[i-1][j] \leq d[i][j] \leq d[i][j+1]d[i−1][j]≤d[i][j]≤d[i][j+1]

使用四边形不等式优化

我们需要证明 w(i,j)w(i,j)w(i,j) 这个二元函数满足区间包含单调性和四边形不等式，才能说明dp满足四边形不等式

  1. 区间包含单调性
    若 l≤l′≤r′≤rl \leq l' \leq r' \leq rl≤l′≤r′≤r 则 w(l′,r′)≤w(l,r)w(l',r') \leq w(l,r)w(l′,r′)≤w(l,r)
  2. 四边形不等式
    若 l≤l′≤r′≤rl \leq l' \leq r' \leq rl≤l′≤r′≤r 则 w(l,r′)+w(l′,r)≤w(l,r)+w(l′+r′)w(l,r')+w(l',r) \leq w(l,r)+w(l'+r')w(l,r′)+w(l′,r)≤w(l,r)+w(l′+r′) 此外还有如下结论帮助我们证明：
  3. 如果 f(l,r)f(l,r)f(l,r) 和 g(l,r)g(l,r)g(l,r) 符合四边形不等式/区间包含单调性，则对于任意 A,B≥0A,B \geq 0A,B≥0 都有 Af(l,r)+Bg(l,r)Af(l,r)+Bg(l,r)Af(l,r)+Bg(l,r) 也符合四边形不等式/区间包含单调性
  4. 如果存在 f(x)f(x)f(x) 和 g(x)g(x)g(x) 使 w(l,r)=f(r)−g(l)w(l,r)=f(r)-g(l)w(l,r)=f(r)−g(l) ，则 w(l,r)w(l,r)w(l,r) 符合四边形恒等式（即总是等号成立的四边形不等式）。如果 f,gf,gf,g 单调递增，那么 www 还符合区间包含单调性
  5. 若 h(x)h(x)h(x) 单调递增且下凸（h′(x)h'(x)h′(x) 单调递增）,w(l,r)w(l,r)w(l,r) 符合四边形不等式和区间包含单调性，则 h(w(l,r))h(w(l,r))h(w(l,r)) 也符合四边形不等式和区间包含单调性

避免证明四边形不等式

将dp最优决策点 d[i][j]d[i][j]d[i][j] 数组打表，如果发现每一行、每一列都具有单调性，那么可以大胆赌一把dp满足四边形不等式

斜率优化

如果转移方程中，dp的当前位置i和准备转移到当前位置的决策点j存在形如 g(i)∗f(j)g(i)*f(j)g(i)∗f(j) 的一项，那么大概率可以使用斜率优化 将转移方程进行整理，由于当前dp到的位置i是固定的，因此所有与i有关的变量（除了 dp[i]dp[i]dp[i] ）都视为常数。将含有 jjj 的变量标记出来，将其移项，使得等式的一边只有一项带有 jjj，其余和 jjj 有关的都移到另一边，这样我们就可以整理得到形如 y=kx+by=kx+by=kx+b 的式子，并且我们要求的 dp[i]dp[i]dp[i] 会在b这个位置。按题目要求，我们最大化/最小化截距。 按照上面的式子，对于每个决策点，我们将其在二维平面上表示为点 (x,y)(x,y)(x,y)。然后做斜率为k的直线并进行平移，使得截距取得最值。kkk 叫做目标斜率。要想快速得到最优决策点，我们需要维护一个凸壳。 一般来说，题目要求最小值，我们维护下凸壳，求最大值，我们维护上凸壳 下凸壳的点集，相邻点连线的斜率是单调递增的。下凸壳的点集，斜率则是单调递减的 按照凸壳方向和目标斜率之间的关系，我们分为下面几类

单调队列维护

下凸壳，目标斜率递增；上凸壳，目标斜率递减 使用单调队列维护 首先我们需要找到最优决策点。这个点和上个点的斜率需要小于目标斜率，和下一个点的斜率需要大于目标斜率。不满足的点直接移出单调队列

int get(int k){
	if(tail==head) return que[head];
	int l=head,r=tail,mid,ans;
	while(l<=r){
		mid=(l+r)>>1;
		if(slope(que[mid],que[mid+1])<=k)){
			ans=mid;
			r=mid-1;
		}else{
			l=mid+1;
		}
	}
	return ans;
}
